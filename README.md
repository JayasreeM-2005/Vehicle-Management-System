Vehicle Management System
A simple web-based application to manage vehicle records. It allows users to add, update, search, and organize vehicle details efficiently.

Features
- Vehicle registration and record management
- Update and delete vehicle information
- Search and filter vehicles
- User-friendly interface

Technologies Used
- HTML
- CSS
- JavaScript
- Python (Flask)
- SQLite Database

 Purpose
This project helps simplify vehicle data management for individuals or organizations.
